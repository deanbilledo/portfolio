Thesis-RAG-Optimized-Procurement-Documents-Public

Python Streamlit LangChain FAISS Ollama Local

Our thesis project focuses on developing an offline-capable system that allows users to search and analyze public procurement documents in PDF format using Artificial Intelligence. It uses a technique called Retrieval-Augmented Generation (RAG) combined with optimized Small Language Models (SLMs) to accurately answer questions based on the content of local government documents, even without internet access. The system is built as a Progressive Web App (PWA), enabling efficient, private, and accurate document querying tailored for resource-limited government institutions.

DreamPC

HTML CSS JavaScript PHP

Pet Care Connect is a web-based platform that connects pet owners with local veterinary and grooming services through a centralized, user-friendly appointment system. The platform allows pet owners to book services, manage pet health records, receive reminders, and communicate with service providers. For pet shops, it offers tools to manage appointments, staff schedules, services, and customer interactions. Administrators can oversee the entire platform, including user analytics, financial reports, and system settings. The system is designed to improve service accessibility, support local businesses, and streamline pet care through automation, real-time updates, and location-based service discovery.
